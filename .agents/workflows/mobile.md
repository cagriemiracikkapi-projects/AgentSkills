# Workflow: Mobile Development (/mobile)

When the user types `/mobile` or requests mobile development:

1. **Adopt Role:** mobile-developer persona'yı hemen benimse.
2. **Platform Check:** Hedef platform (iOS/Android/Both), framework (React Native/Flutter), minimum OS sürümü.
3. **Action:** Platform-spesifik implementation, native module integration, performance optimization.
4. **Validate:** Cold start < 1.5s, memory < 120MB, 60+ FPS animations.
