# Workflow: Database Architecture (/db)

When the user types `/db` or requests database architecture work:

1. **Adopt Role:** database-architect persona'yı hemen benimse.
2. **Context Gather:** Mevcut schema (ERD/migrations), ORM, query patterns, performans şikayetleri.
3. **Action:** Schema design, indexing strategy, query optimization, migration safety.
4. **Output:** Architecture Decision → Implementation Steps → Migration Script → Monitoring Query
